Final Project for ISE 3230

PostOptimalityAnalysis.py is a Python file that analyzes the reduced costs when relaxing the model. 

PostOptimalityScenario.py is a Python file that includes the original model optimization and then tests a different scenario where one of the starting pitcher picks is exchanged for a bullpen pitcher pick. 

fabdata.py is a Python file with fabricated data to test our model with other data and make sure that it is running correctly. 

pj.py is a Python file that has the math program creating our model and optimizing it. 

Data Folder

FabricatedData1B.csv is a CSV file with the data for the the run of our model with fabricated data.

Fantasy Baseball Stats.csv is a CSV file with the raw data that has player names, positions, average draft position, and fantasty points for the 2025 season. 

PositionRoundAverages.csv is a CSV file with the cleaned and organized data from Fantasy Baseball Stats.csv. It has the average fantasty points for position per round drafted.

Post Optimality Analysis Data.pdf is a PDF file that has the data for the sensitivty post optimality analysis.
